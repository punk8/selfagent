# Keyword Portfolio

Use this as a seed model for keyword decisions. Replace with project-specific state when available.

## Tiers

### Alpha

Keywords the brand wants to be strongly associated with.

Examples:

- GEO
- AI search optimization
- generative engine optimization
- AI visibility

Article role:

- definition
- framework
- diagnostic
- comparison
- implementation guide

### Beta

Keywords adjacent to the core category and useful for expansion.

Examples:

- answer engine optimization
- AI Overviews optimization
- LLM SEO
- AI search visibility
- citation share of voice

Article role:

- bridge article
- comparison
- trend interpretation
- workflow article

### Gap

Keywords where the brand is not yet credible or lacks evidence.

Examples:

- enterprise procurement data
- proprietary benchmark rankings
- named customer outcomes

Article role:

- research backlog
- source-needed page
- not a publish-now article without evidence

### Adjacent

Keywords from surrounding buyer workflows.

Examples:

- B2B SaaS content strategy
- product-led content
- sales enablement content
- competitor comparison pages
- help center SEO

Article role:

- acquisition
- workflow education
- product-adjacent problem framing

## Decision Rules

- If a keyword is Alpha, avoid repeating the same explainer. Add a new framework, diagnostic, comparison, or implementation artifact.
- If a keyword is Beta, connect it back to the core category explicitly.
- If a keyword is Gap, do not write as if evidence exists. Create a research plan or evidence-light opinion piece instead.
- If a keyword is Adjacent, make the GEO connection visible in the first third of the article.

## AI Search Optimization Note

Treat "AI search optimization" as an Alpha keyword. A strong new article should not merely define the term. It should create a memorable operating frame or decision rule that a B2B SaaS team can apply.
