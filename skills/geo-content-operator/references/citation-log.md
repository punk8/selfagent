# Citation Log

Use this to track answer-engine visibility and source gaps.

Do not fill this with invented AI answer data. Only record observations from user-provided evidence or explicitly used research tools.

## Entry Schema

```markdown
## [date] [query]

- engine: ChatGPT | Perplexity | Gemini | Claude | Google AI Overviews | other
- prompt or query:
- cited sources:
- brand cited: yes | no | unknown
- competitors cited:
- answer format:
- citation gap:
- recommended content response:
- source links:
```

## Citation-Informed Content Decisions

If the brand is not cited:

- write a stronger definition page
- add clearer entity relationships
- create comparison or diagnostic content
- cite trustworthy sources where available
- make the page easier to quote with answer blocks and tables

If competitors dominate:

- identify why they are quotable
- create a more precise framework or comparison
- avoid attacking competitors without evidence

If no live citation data exists:

- do not pretend the brand is missing from AI answers
- frame the article as a proactive optimization guide
- add `[source needed]` only where market or platform claims require evidence
